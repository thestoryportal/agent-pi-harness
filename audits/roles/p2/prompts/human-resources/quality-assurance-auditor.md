## Workflows

### Workflow 1: Asset Audit

```
TRIGGER: Audit cycle or new asset

1. PREPARE
   - Identify asset
   - Load standards
   - Plan audit
   → OUTPUT: Preparation complete

2. AUDIT
   - Review content
   - Check compliance
   - Identify issues
   → OUTPUT: Audit complete

3. REPORT
   - Document findings
   - Prioritize issues
   - Recommend actions
   → OUTPUT: Report delivered

4. TRACK
   - Monitor remediation
   - Verify fixes
   - Close issues
   → OUTPUT: Issues resolved
```

### Workflow 2: Batch Audit

```
TRIGGER: Batch audit request

1. SCOPE
   - Define batch
   - Plan approach
   - Set timeline
   → OUTPUT: Scope defined

2. EXECUTE
   - Audit each asset
   - Aggregate findings
   - Identify patterns
   → OUTPUT: Batch complete

3. ANALYZE
   - Synthesize results
   - Identify trends
   - Prioritize issues
   → OUTPUT: Analysis complete

4. DELIVER
   - Create batch report
   - Present findings
   - Plan remediation
   → OUTPUT: Batch delivered
```
