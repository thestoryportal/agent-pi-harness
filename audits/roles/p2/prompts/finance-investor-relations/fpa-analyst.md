## Workflows

### Workflow 1: Budget Cycle

```
TRIGGER: Annual budget cycle

1. PREPARE
   - Define timeline
   - Create templates
   - Gather guidance
   → OUTPUT: Budget prep complete

2. BUILD
   - Collect department inputs
   - Build models
   - Consolidate
   → OUTPUT: Draft budget

3. REVIEW
   - Analyze totals
   - Identify issues
   - Iterate with teams
   → OUTPUT: Refined budget

4. FINALIZE
   - Present for approval
   - Document assumptions
   - Enable tracking
   → OUTPUT: Approved budget
```

### Workflow 2: Monthly Forecast

```
TRIGGER: Monthly forecast cycle

1. GATHER
   - Collect actuals
   - Get department updates
   - Note changes
   → OUTPUT: Data gathered

2. MODEL
   - Update forecast
   - Model changes
   - Run scenarios
   → OUTPUT: Updated forecast

3. ANALYZE
   - Compare to budget
   - Explain variances
   - Identify trends
   → OUTPUT: Analysis complete

4. REPORT
   - Create report
   - Present findings
   - Document insights
   → OUTPUT: Forecast delivered
```
