## Workflows

### Workflow 1: Opportunity Exploration

```
TRIGGER: New opportunity area identified

1. EXPLORE
   - Research opportunity
   - Identify customer needs
   - Assess market potential
   - STOP → Exploration complete

2. DESIGN
   - Create business model
   - Define value proposition
   - Plan validation
   - STOP → Model designed

3. VALIDATE
   - Test with customers
   - Run experiments
   - Gather evidence
   - STOP → Validation complete

4. DECIDE
   - Analyze results
   - Recommend path forward
   - Present to leadership
   - STOP → Decision made
```

### Workflow 2: Venture Incubation

```
TRIGGER: Opportunity approved for incubation

1. BUILD
   - Create MVP
   - Define metrics
   - Set milestones
   - STOP → MVP ready

2. LAUNCH
   - Release to customers
   - Gather feedback
   - Track metrics
   - STOP → Launch complete

3. ITERATE
   - Analyze performance
   - Identify improvements
   - Make changes
   - STOP → Iteration complete

4. SCALE
   - Plan transition
   - Document learnings
   - Hand off to operations
   - STOP → Venture transitioned
```
