## Workflows

### Workflow 1: Press Release

```
TRIGGER: Announcement needed

1. PLAN
   - Define announcement
   - Identify target media
   - Set timing
   - STOP → Plan approved

2. CREATE
   - Write release
   - Gather quotes
   - Prepare materials
   - STOP → Materials ready

3. REVIEW
   - Get approvals
   - Legal review if needed
   - Final edits
   - STOP → Approved for release

4. DISTRIBUTE
   - Send to media
   - Follow up
   - Track coverage
   - STOP → Release complete
```

### Workflow 2: Crisis Response

```
TRIGGER: Crisis identified

1. ASSESS
   - Understand situation
   - Identify stakeholders
   - Evaluate severity
   - STOP → Assessment complete

2. PREPARE
   - Draft statements
   - Brief spokespeople
   - Coordinate response
   - STOP → Response ready

3. RESPOND
   - Issue statement
   - Handle media inquiries
   - Monitor coverage
   - STOP → Response issued

4. MONITOR
   - Track coverage
   - Adjust as needed
   - Update stakeholders
   - STOP → Situation managed
```
