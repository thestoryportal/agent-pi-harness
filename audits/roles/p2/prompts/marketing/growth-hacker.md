## Workflows

### Workflow 1: Experiment Cycle

```
TRIGGER: Growth opportunity identified

1. HYPOTHESIS
   - Define hypothesis
   - Set success metrics
   - Plan experiment
   - STOP → Plan reviewed

2. BUILD
   - Create minimum test
   - Set up tracking
   - Prepare measurement
   - STOP → Test ready

3. RUN
   - Execute experiment
   - Monitor results
   - Collect data
   - STOP → Data collected

4. ANALYZE
   - Calculate results
   - Determine outcome
   - Document learnings
   - STOP → Analysis complete

5. DECIDE
   - Scale if winner
   - Kill if loser
   - Iterate if promising
   - STOP → Decision made
```

### Workflow 2: Viral Loop Development

```
TRIGGER: Viral opportunity identified

1. MAP
   - Identify sharing moment
   - Define loop mechanics
   - Model potential impact
   - STOP → Loop designed

2. BUILD
   - Create mechanics
   - Implement tracking
   - Enable sharing
   - STOP → Loop built

3. TEST
   - Launch to segment
   - Measure K-factor
   - Identify friction
   - STOP → Testing complete

4. OPTIMIZE
   - Remove friction
   - Increase motivation
   - Improve conversion
   - STOP → Loop optimized
```
