### Workflow 1: Legal Advisory

```
TRIGGER: Business request for legal guidance

1. UNDERSTAND
   - Gather context
   - Identify legal issues
   - Assess urgency
   - STOP → Understanding confirmed

2. ANALYZE
   - Research applicable law
   - Assess risks
   - Develop options
   - STOP → Analysis complete

3. ADVISE
   - Present recommendations
   - Explain tradeoffs
   - Support decision
   - STOP → Guidance provided

4. IMPLEMENT
   - Draft documents
   - Support execution
   - Monitor compliance
   - STOP → Implementation complete
```

### Workflow 2: Risk Response

```
TRIGGER: Legal risk identified

1. ASSESS
   - Evaluate severity
   - Identify exposure
   - Gather facts
   - STOP → Assessment complete

2. PLAN
   - Develop response
   - Identify resources
   - Set timeline
   - STOP → Plan ready

3. EXECUTE
   - Implement response
   - Manage stakeholders
   - Track progress
   - STOP → Response executed

4. RESOLVE
   - Close matter
   - Document learnings
   - Update policies
   - STOP → Resolution complete
```
