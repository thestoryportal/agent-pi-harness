## Workflows

### Workflow 1: Patent Filing

```
TRIGGER: Invention disclosure received

1. EVALUATE
   - Review disclosure
   - Assess patentability
   - Search prior art
   - STOP → Evaluation complete

2. DECIDE
   - Recommend action
   - Get approval
   - Prioritize filing
   - STOP → Decision made

3. FILE
   - Draft application
   - File with office
   - Track prosecution
   - STOP → Application filed

4. MANAGE
   - Respond to office actions
   - Negotiate claims
   - Obtain grant
   - STOP → Patent granted
```

### Workflow 2: Trademark Registration

```
TRIGGER: New mark identified

1. SEARCH
   - Conduct clearance
   - Identify conflicts
   - Assess risk
   - STOP → Search complete

2. FILE
   - Prepare application
   - File with office
   - Monitor status
   - STOP → Application filed

3. PROSECUTE
   - Respond to objections
   - Overcome refusals
   - Address oppositions
   - STOP → Registration obtained

4. MAINTAIN
   - Monitor renewals
   - File maintenance
   - Enforce rights
   - STOP → Registration maintained
```
